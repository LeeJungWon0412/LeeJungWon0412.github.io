📖 SVG 라인 애니메이션 설명 보기 ▼

이 도구는 SVG(Scalable Vector Graphics)의 `stroke-dasharray`와 `stroke-dashoffset` 속성을 이용해 선이 따라 그려지는 듯한 애니메이션 효과를 만듦.

아이콘 라이브러리 등에서 받은 SVG 코드를 붙여넣고 '애니메이션 생성' 버튼을 누르삼.

생성된 CSS 코드를 복사하여 자신의 SVG 코드와 함께 사용하면 됨.
