이 도구는 CSS의 `filter` 속성을 사용하여 이미지에 다양한 시각적 효과를 적용함.

포토샵 없이도 웹에서 직접 이미지의 느낌을 바꿀 수 있삼.

각 슬라이더를 조절하여 밝기, 대비, 채도, 흑백, 블러 등 여러 효과를 조합하고, 생성된 코드를 복사하여 자신의 프로젝트에 활용해보삼.
