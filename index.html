필요할 때마다 찾아 쓰기 귀찮아서 내가 쓰려고 만든 사이트

이게 뭐예요? ▼

이정원의 도구 모음집은 사진 압축, 글자 수 세기, 이미지 파일 형식 변환, 단위 변환 등 간단하고 실용적인 웹 도구들을 제공함.

모든 도구는 사용자 기기에서 직접 처리되어 개인 정보 유출 걱정 없이 안전하게 사용할 수 있삼.

앞으로도 여러 가지 기능들을 계속 추가할 거임.

도구 목록

  • 🖼️ 사진 압축기
  • 📝 글자 수 세기
  • 🖼️ 이미지 파일 형식 변환기
  • 📏 단위 변환기
  • 🗓️ 날짜 계산기
  • 🎨 CSS 그림자 생성기
  • 🎨 색상 코드 변환기
  • 🎂 만 나이, 띠, 별자리 계산기
  • 💕 종합 궁합 테스트
  • 📱 QR코드 생성기
  • 💵 환율 계산기
  • 💰 대출 이자 계산기
  • 🎱 로또 번호 생성기
  • ✨ 이모지/특수문자 복사
